# 📘 Interview Notes

Цей репозиторій є структурованим довідником для підготовки до технічних співбесід у сфері frontend та WordPress розробки.

## 📚 Опис

Матеріали організовані у вигляді тематичних розділів і підрозділів, що охоплюють як базові, так і просунуті теми. Кожен файл присвячений окремому питанню або концепції, які часто зустрічаються на співбесідах.

Репозиторій включає:

* основи та поглиблені теми frontend розробки (HTML, CSS, JavaScript)
* сучасні бібліотеки та фреймворки (React, Next.js)
* екосистему WordPress (core, теми, плагіни, API)
* алгоритми та структури даних
* основи системного дизайну
* behavioral та HR питання

## 🧠 Призначення

* систематизація технічних знань
* підготовка до співбесід
* швидкий пошук відповідей на типові питання
* довідковий матеріал для повторення

## 📁 Структура

Репозиторій побудований за принципом вкладеної ієрархії:

* розділи → напрями (frontend, wordpress, computer-science тощо)
* підрозділи → конкретні теми
* файли → окремі питання або концепції

Це забезпечує зручну навігацію та можливість масштабування.

## 📌 Формат матеріалів

Кожен файл може містити:

* формулювання питання
* коротку або розгорнуту відповідь
* приклади коду
* пояснення ключових моментів
* додаткові (follow-up) питання

## ⚙️ Використання

Репозиторій можна використовувати як:

* чекліст для підготовки
* довідник перед співбесідою
* базу знань для системного повторення

---

> Матеріали можуть доповнюватися та оновлюватися відповідно до змін у технологіях та вимогах ринку.
