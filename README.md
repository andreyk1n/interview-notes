# 📘 Developer Interview Preparation

Структурована база знань із запитаннями та відповідями для підготовки до технічних співбесід у frontend та WordPress розробці.

## 📚 Зміст

Репозиторій охоплює такі напрями:

* **Frontend fundamentals** — HTML, CSS, JavaScript (базові та просунуті теми, робота браузера)
* **React & Next.js** — ключові концепції, хуки, підходи до рендерингу та оптимізації
* **WordPress** — core, теми, плагіни, WooCommerce, Gutenberg, REST API
* **Computer Science** — алгоритми та структури даних
* **System Design** — основи архітектури та масштабування
* **Behavioral / HR** — типові питання зі співбесід

## 🎯 Призначення

* Систематизація знань
* Підготовка до технічних співбесід
* Швидкий доступ до ключових тем та понять

## 🧠 Структура матеріалів

Кожна тема оформлена як окремий файл і містить:

* запитання
* відповідь
* приклади (за потреби)
* поширені помилки
* додаткові запитання

## 📁 Організація

Матеріали згруповані за категоріями та підтемами для зручної навігації та масштабування.
