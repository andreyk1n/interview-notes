# CSS Typography / Font

## Визначення

**Typography в CSS** — це набір властивостей, які керують зовнішнім виглядом і читабельністю тексту на сторінці. Вони визначають:

* вибір шрифтів
* розміри та інтервали
* вирівнювання та перенос
* візуальну ієрархію і доступність

Основна властивість: `font`

---

## font-family

```css
p {
    font-family: Arial, Helvetica, sans-serif;
}
```

Визначає перелік шрифтів, які має спробувати використати браузер для відображення тексту.
Якщо перший шрифт недоступний, браузер переходить до наступного.

---

### Типи шрифтів

```css
serif
sans-serif
monospace
cursive
fantasy
```

Це системні категорії шрифтів:
* `serif` — зі зарубками
* `sans-serif` — без зарубок
* `monospace` — фіксована ширина символів
* `cursive` — рукописні форми
* `fantasy` — декоративні

---

### Fallback

```css
font-family: "Inter", Arial, sans-serif;
```

Якщо Inter недоступний → використовується Arial → потім системний шрифт типу sans-serif.
Це важливо для надійності та швидкості завантаження.

---

## Web Fonts (@font-face)

```css
@font-face {
    font-family: "MyFont";
    src: url("/fonts/myfont.woff2") format("woff2");
}
```

Оголошує шрифт, який завантажується з сервера. `font-family` задає ім'я для подальшого використання.

---

### Формати

* woff2 — сучасний, стиснутий, кращий для вебу
* woff — сумісний зі старішими браузерами
* ttf — менш оптимізований, може бути більшим

---

### Кращі практики

```css
src: url("font.woff2") format("woff2"),
     url("font.woff") format("woff");
```

Спочатку підключаємо оптимальний формат, потім резервний.

---

## font-size

```css
p {
    font-size: 16px;
}
```

Визначає розмір шрифту для елемента. Від цього залежать загальна читабельність і масштаб тексту.

---

### Одиниці

| Одиниця | Опис                                  |
| ------- | ------------------------------------- |
| px      | фіксований розмір, точно в пікселях   |
| em      | відносно розміру шрифту батьківського елемента |
| rem     | відносно розміру шрифту кореневого `html` |
| %       | відносно розміру шрифту батьківського елемента |

---

### Кращі практики

```css
html { font-size: 16px; }
p { font-size: 1rem; }
```

Використовувати `rem` для узгоджених пропорцій і кращої масштабованості.

---

## font-weight

```css
font-weight: 400;
font-weight: bold;
```

Визначає товщину шрифту. Браузери використовують набір доступних варіантів для конкретного шрифту.

---

| Значення | Опис    |
| -------- | ------- |
| 100–900  | тонкий до жирного               |
| normal   | 400                            |
| bold     | 700                            |

---

Variable fonts можуть мати будь-які значення між 100 і 900, якщо шрифт це підтримує.

---

## font-style

```css
font-style: italic;
```

Визначає стиль начерку:
* `normal` — звичайний
* `italic` — курсивний
* `oblique` — похилий

---

| Значення |
| -------- |
| normal   |
| italic   |
| oblique  |

---

## font-variant

```css
font-variant: small-caps;
```

Керує альтернативними варіантами начерку, наприклад малими прописними буквами.
Рідко використовується, але корисно для декоративних або типографічних ефектів.

---

## line-height

```css
line-height: 1.5;
```

Визначає висоту рядка і відстань між базовими лініями сусідніх рядків.
Впливає на читабельність і візуальний ритм тексту.

---

### Кращі практики

```css
line-height: 1.4–1.6;
```

Задаємо без одиниць, щоб значення було відносним до розміру шрифту.

---

## letter-spacing

```css
letter-spacing: 1px;
```

Встановлює додатковий проміжок між символами.
Корисно для заголовків, де потрібно покращити читабельність.

---

## word-spacing

```css
word-spacing: 4px;
```

Встановлює відстань між словами.
Використовується рідше, але може допомогти вирівняти текст або поліпшити читабельність.

---

## text-transform

```css
text-transform: uppercase;
```

Перетворює регістр тексту:
* `uppercase` — великі літери
* `lowercase` — малі літери
* `capitalize` — перша літера кожного слова

---

| Значення   |
| ---------- |
| uppercase  |
| lowercase  |
| capitalize |

---

## text-align

```css
text-align: center;
```

Вирівнює текст по горизонталі всередині контейнера:
* `left`
* `right`
* `center`
* `justify`

---

## text-decoration

```css
text-decoration: underline;
```

Додає декоративну лінію до тексту:
* підкреслення
* перекреслення
* надрядкове підкреслення

---

### Розширений варіант

```css
text-decoration: underline dashed red;
```

Контролює стиль, товщину і колір декорації.

---

## text-overflow

```css
text-overflow: ellipsis;
```

Визначає, як обрізати текст, що не вміщується в блок.
Працює при:

```css
white-space: nowrap;
overflow: hidden;
```

---

## white-space

```css
white-space: nowrap;
```

Керує обробкою пробілів і переносів:
* `normal` — звичайний перенос
* `nowrap` — без переносу
* `pre` — зберігає пробіли як у коді

---

| Значення | Опис         |
| -------- | ------------ |
| normal   | перенос      |
| nowrap   | без переносу |
| pre      | як у коді    |

---

## overflow-wrap / word-break

```css
word-break: break-word;
```

Контролює перенос довгих слів, щоб текст не виходив за межі блоку.
Це важливо для адаптивного дизайну.

---

## font shorthand

```css
font: italic 700 16px/1.5 "Inter", sans-serif;
```

Скорочено задає кілька властивостей одночасно:
* стиль
* товщину
* розмір
* міжрядковий інтервал
* сімейство шрифтів

---

### Порядок

```text
style → weight → size/line-height → family
```

Потрібно дотримуватися порядку, щоб шорткейс правильно розібрався.

---

## Anti-aliasing (не стандарт)

```css
-webkit-font-smoothing: antialiased;
```

Надає оптимізоване згладжування шрифтів у WebKit-браузерах для кращого вигляду на екрані.

---

## Variable Fonts

```css
font-variation-settings: "wght" 500;
```

Дозволяє змінювати вісь вартостей шрифту, наприклад вагу, ширину, нахил.
Один файл може замінити багато звичайних файлів шрифтів.

---

## Практичні приклади

---

### Базова типографіка

```css
body {
    font-family: "Inter", sans-serif;
    font-size: 16px;
    line-height: 1.5;
}
```

Встановлює базовий шрифт, розмір і міжрядковий інтервал для всього документа.

---

### Заголовок

```css
h1 {
    font-size: 2rem;
    font-weight: 700;
}
```

Визначає помітний стиль для заголовків.

---

### Обрізка тексту

```css
.text {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}
```

Забезпечує елегантне обрізання тексту в обмеженому просторі.

---

## Accessibility

* не використовувати занадто маленький текст
* достатній `line-height`
* контраст між текстом і фоном
* вибирати читабельні шрифти

---

## Performance

* використовувати `woff2`
* `preload` шрифтів
* не підключати багато варіантів
* мінімізувати кількість зовнішніх файлів

---

* що таке fallback?

* як працює text-overflow?

## Часті помилки

* `px` замість `rem`
* відсутній fallback
* занадто багато шрифтів
* неправильний `line-height`

---

## Кращі практики

* використовувати `rem`
* задавати базовий `font` у `body`
* використовувати fallback
* оптимізувати шрифти

---

## Висновок

Typography:

* критична для UX
* впливає на читабельність
* важлива для accessibility і performance
