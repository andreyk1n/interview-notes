# CSS Borders / Border Radius / Outline

## Визначення

Цей набір властивостей відповідає за візуальні межі елементів:

* `border` - визначає рамку навколо елемента
* `border-radius` - визначає заокруглення кутів
* `outline` - визначає зовнішній контур, який не бере участі в моделі коробки

---

# 1. `border`

## Основи

```css
.box {
    border: 1px solid black;
}
```

Властивість `border` створює видиму лінію навколо елемента, відділяючи його від навколишнього вмісту. Вона складається з товщини, стилю та кольору і впливає на розміри елемента, додаючи простір до його габаритів, якщо не використано `box-sizing: border-box`.

---

## Синтаксис скорочення

```css
border: width style color;
```

Цей скорочений запис задає всі три характеристики рамки одразу: товщину (width), вигляд (style) та колір (color). Застосовується до всіх чотирьох сторін, якщо не перевизначено окремими властивостями.

---

### Приклад

```css
border: 2px dashed red;
```

У цьому прикладі рамка має товщину 2 пікселі, пунктирний стиль і червоний колір, створюючи візуальне розділення з переривчастою лінією.

---

## Окремі властивості

```css
border-width: 2px;
border-style: solid;
border-color: red;
```

Це дозволяє задавати кожен аспект окремо. `border-width` контролює товщину, `border-style` визначає тип лінії (наприклад, solid, dashed), а `border-color` задає колір, який може бути будь-яким дійсним значенням CSS.

---

## Сторони рамки

```css
border-top: 1px solid red;
border-right: 1px solid blue;
border-bottom: 1px solid green;
border-left: 1px solid black;
```

Ці властивості націлені на конкретні сторони елемента, дозволяючи встановлювати різні рамки для кожного краю. Це корисно для створення напрямкового акценту або розділення областей всередині елемента.

---

## Типи стилю рамки

| Value  | Description     |
| ------ | --------------- |
| solid  | continuous line |
| dashed | broken line     |
| dotted | dotted line     |
| double | two parallel lines |
| none   | no border      |

Властивість `border-style` визначає візуальний візерунок лінії рамки, впливаючи на вигляд краю. `none` повністю видаляє рамку.

---

## Рамка і box-sizing

```css
box-sizing: border-box;
```

Коли `box-sizing` встановлено в `border-box`, товщина рамки включається в загальну ширину та висоту елемента, що запобігає збільшенню елемента при додаванні рамки. Це важливо для адаптивної верстки.

---

## Рамка і currentColor

```css
.box {
    color: red;
    border: 2px solid currentColor;
}
```

`currentColor` використовує колір тексту елемента для рамки, забезпечуючи узгодженість. Якщо колір тексту змінюється, колір рамки автоматично оновлюється.

---

## border-image (додатково)

```css
border-image: url(border.png) 30 stretch;
```

Ця властивість дозволяє використовувати зображення як рамку, розрізаючи його на частини та розтягуючи або повторюючи їх. Це дає творчі візуальні ефекти, але потребує ретельної підготовки зображення.

---

# 2. `border-radius`

## Визначення

```css
.box {
    border-radius: 10px;
}
```

Властивість `border-radius` заокруглює кути рамки (і фон) елемента, створюючи більш м’які, сучасні форми. Вона впливає на візуальний контур без зміни макету елемента.

---

## Синтаксис скорочення

```css
border-radius: 10px 20px 30px 40px;
```

```text
top-left -> top-right -> bottom-right -> bottom-left
```

Це задає різні радіуси для кожного кута за годинниковою стрілкою, дозволяючи створювати асиметричні заокруглення для нестандартних форм.

---

## Окремі кути

```css
border-top-left-radius: 10px;
border-bottom-right-radius: 20px;
```

Ці властивості впливають на конкретні кути, даючи точний контроль над тим, які частини елемента заокруглені.

---

## Коло

```css
.circle {
    width: 100px;
    height: 100px;
    border-radius: 50%;
}
```

Встановлення `border-radius` в 50% для квадратного елемента створює ідеальне коло, оскільки радіус дорівнює половині сторони.

---

## Еліпс

```css
border-radius: 50% / 30%;
```

Це створює еліптичну форму, задаючи окремо горизонтальний і вертикальний радіуси, що корисно для овальних дизайнів.

---

## Важливі зауваження

* Взаємодіє з `overflow` для обрізання вмісту.
* Впливає на обрізання фону.
* Може обрізати перетікання дочірніх елементів.

Ця властивість впливає на те, як фон та вміст відображаються на краях, часто використовується разом з `overflow: hidden` для охайного вигляду.

---

## border-radius і overflow

```css
.box {
    border-radius: 10px;
    overflow: hidden;
}
```

Поєднання цих властивостей гарантує, що заокруглені кути обрізають будь-який вміст або тінь, які виходять за межі, зберігаючи чистий вигляд.

---

# 3. `outline`

## Визначення

```css
.box {
    outline: 2px solid red;
}
```

Властивість `outline` малює лінію навколо елемента зовні його рамки, переважно використовується для індикаторів фокусу. Вона не впливає на макет і не входить у розрахунок моделі коробки.

---

## Відмінність від рамки

| Border              | Outline |
| ------------------- | ------- |
| participates in box-model | no     |
| takes up space      | no     |
| can be per side     | no     |
| affects layout      | no     |

На відміну від `border`, `outline` малюється поза розмірами елемента і не може бути заданий для окремих сторін, що робить його підходящим для ненав’язливих підсвічувань.

---

## Скорочений запис

```css
outline: width style color;
```

Схоже на `border`, задає товщину, стиль і колір в одному оголошенні, але доступні стилі обмеженіші, ніж для рамок.

---

## outline-offset

```css
outline-offset: 4px;
```

Це створює простір між краєм елемента та контуром, дозволяючи кращу візуальну відокремленість без впливу на макет.

---

## Доступність (дуже важливо)

```css
button:focus {
    outline: 2px solid blue;
}
```

Контури важливі для навігації з клавіатури, підкреслюючи сфокусовані елементи для користувачів, які покладаються на допоміжні технології.

---

НЕ ВАРТО РОБИТИ:

```css
outline: none;
```

Це порушує доступність, видаляючи індикатори фокусу і йде врозріз із веб-стандартами.

---

## Стилі фокусу

```css
:focus-visible {
    outline: 2px solid blue;
}
```

Цей псевдоклас застосовує контури лише тоді, коли фокус видно (наприклад, при навігації з клавіатури), уникаючи небажаних стилів при кліку мишею.

---

# Рамка проти контуру

| Property          | Border | Outline |
| ----------------- | ------ | ------- |
| affects layout    | yes    | no      |
| included in size  | yes    | no      |
| can be per sides  | yes    | no      |
| usage             | UI     | focus   |

Рамка змінює простір елемента, тоді як контур є суто декоративним і не впливає на розмітку.

---

# Практичні приклади

---

## Кнопка

```css
button {
    border: 1px solid #ccc;
    border-radius: 8px;
}
```

Додає легку рамку та заокруглені кути для сучасного вигляду кнопки.

---

## Картка

```css
.card {
    border: 1px solid #eee;
    border-radius: 12px;
}
```

Створює контейнер з рамкою та м’якими кутами, що часто використовується в макетах карток.

---

## Фокус на полі введення

```css
input:focus {
    outline: 2px solid blue;
}
```

Забезпечує чіткий індикатор фокусу для полів форм, покращуючи зручність користування.

---

## Аватар

```css
.avatar {
    border-radius: 50%;
}
```

Округлює зображення в коло, ідеально для профільних фото.

* Як створити коло?

---

# Поширені помилки

* Використання `outline: none`
* Забуття про `box-sizing`
* Неправильне розуміння різниці між border та outline
* Неправильне використання скороченого запису

---

# Кращі практики

* Використовуйте border для елементів інтерфейсу
* Використовуйте outline для станів фокусу
* Не вимикайте outline
* Використовуйте border-radius для сучасного інтерфейсу

---

## Висновок

* `border` - визначає рамку
* `border-radius` - визначає форму
* `outline` - визначає підсвічування доступності

Разом вони формують візуальні межі елемента.
