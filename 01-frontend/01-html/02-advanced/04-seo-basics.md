# SEO Basics (Search Engine Optimization)

## Визначення

**SEO (Search Engine Optimization)** — це набір практик для покращення видимості вебсайту в пошукових системах (Google, Bing тощо).

Мета:

* підвищити позиції у видачі
* збільшити органічний трафік
* покращити якість контенту

---

## Типи SEO

### 1. On-page SEO

Оптимізація всередині сторінки:

* HTML структура
* контент
* метадані

---

### 2. Off-page SEO

Зовнішні фактори:

* беклінки
* авторитет домену

---

### 3. Technical SEO

Технічні аспекти:

* швидкість
* індексація
* структура сайту

---

## Основні фактори SEO

---

## 1. HTML структура

### Title

```html id="z9a9he"
<title>Page Title</title>
```

Один з найважливіших факторів

---

### Meta Description

```html id="pnj0mi"
<meta name="description" content="Page description">
```

Впливає на CTR (кліки)

---

### Заголовки (H1-H6)

```html id="sl4u6j"
<h1>Main Title</h1>
<h2>Section</h2>
```

Правила:

* один `<h1>`
* логічна ієрархія

---

## 2. Контент

Важливо:

* унікальний
* релевантний
* структурований

---

### Ключові слова

* природне використання
* без keyword stuffing

---

## 3. Семантичний HTML

```html id="o6p0nt"
<article>
<section>
<nav>
```

Допомагає пошуковим системам

---

## 4. Зображення

```html id="yz0u3n"
<img src="img.jpg" alt="description">
```

alt:

* описує зображення
* впливає на SEO

---

## 5. URL структура

Добре:

```id="9h9lpt"
/blog/seo-basics
```

Погано:

```id="tchv6h"
/page?id=123
```

---

## 6. Внутрішні посилання

```html id="ks9gpo"
<a href="/blog">Blog</a>
```

Покращують навігацію та SEO

---

## 7. Mobile-friendly

```html id="nnkw5o"
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Mobile-first indexing

---

## 8. Швидкість (Performance)

Впливає на ранжування:

* оптимізація зображень
* lazy loading
* кешування

---

## 9. Open Graph

```html id="ztl9t1"
<meta property="og:title" content="Title">
<meta property="og:image" content="image.jpg">
```

Для соцмереж

---

## 10. Robots

```html id="x9t1aa"
<meta name="robots" content="index, follow">
```

---

## 11. Sitemap

XML файл:

* допомагає індексації

---

## 12. Canonical

```html id="j0e2mj"
<link rel="canonical" href="https://example.com/page">
```

Уникає дублювання контенту

---

## 13. Structured Data (Schema)

```html id="b6d6je"
<script type="application/ld+json">
{
    "@context": "https://schema.org"
}
</script>
```

Rich snippets

* дубль контенту

## Best Practices

* писати унікальний title і description
* використовувати семантичний HTML
* оптимізувати швидкість
* робити mobile-friendly
* використовувати внутрішні посилання
* уникати дублювання

---

## Висновок

SEO:

* поєднання техніки + контенту
* впливає на видимість сайту
* критично для продуктів і бізнесу
